排序算法的稳定性及其汇总

同样值得个体之间，如果不因为排序而改变相对次序，就是这个排序是有稳定性得；否则就没有。

```js
const arr = [1, 2, 1, 3, 3, 2][
  // 排序之后
  (1, 1, 2, 2, 3, 3)
]
// 并且原来的0位置的1，排序之后还是排在原来下标为2的1的前面
```

不具备稳定性的排序：

- 选择排序
- 快速排序
- 堆排序

具备稳定性的排序：

- 冒泡排序
- 插入排序
- 归并排序
- 一切桶排序思想下的排序

暂不存在时间复杂度`O(NlogN)`，额外空间复杂度`O(1)`，又稳定的排序。
